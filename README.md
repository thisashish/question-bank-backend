# Question Bank Backend

A complete Question Bank Management System built with Node.js, Express, TypeScript, and MongoDB.

## Features

-  Create, Read, Update, Delete Questions
-  Multiple Question Types (MCQ, Match Following, etc.)
-  Advanced Filtering and Search
-  Swagger API Documentation
-  TypeScript Support
-  MongoDB Integration
